
# Used pins

## PORT A

  * PA9  - UART TX (traces)

  * PA11 - USB DM
  * PA12 - USB DP

  * PA13 - SWDIO
  * PA14 - SWCLK

## PORT C

  * PC13 - LED (blinking)

# Used clocks

## USB

  * USBCLK  - 48 MHz

## AHB

  * HCLK (AHB, core, mem, DMA) - 48 MHz
  * SYSTIMCLK (System timer) - 48/8 = 6 MHz
  * FCLK - 48 MHz

## APB1

  * PCLK1 (APB1 periph) - 24 MHz

## APB2

  * PCLK2 (APB2 periph) - 48 MHz
  * ADCCLK - 48/8 = 6 MHz
